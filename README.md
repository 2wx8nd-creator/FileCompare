# File Compare
## 개요 - C# 프로그래밍학습
-1줄소개: 서로 다른 파일, 폴더의 비교 프로그램  
-C#, .NET Windows Forms, Visual Studio, GitHub

	-구현 기능 : 파일 및 폴더 선택 기능  
## 과제2 
![실행화면](img/screenshot-2.png)  
*구현 기능  
	1.선택된 폴더의 파일을 리스트 뷰에 출력  
	2.리스트 뷰에 출력된 파일의 상태에 따라 다른 색상으로 표시  
		  -같은 파일 : 파란색  
		  -수정시간의 차이 : 빨간색(최신), 회색(이전)  
		  -없는 파일 : 보라색
